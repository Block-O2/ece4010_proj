# Project Summary

This project predicts whether a dog or cat entering the Austin Animal Center will be adopted within 30 days, using only information available at intake time. The repository uses two public datasets, Austin Animal Center Intakes and Outcomes, and constructs the main binary target `fast_adoption_30d`. Because repeated `Animal ID` values appear in both tables, intake and outcome records are matched using a documented chronological pairing rule: sort each animal’s records by time and pair the `k`-th intake with the `k`-th outcome. A fallback target `adoption` is also kept for comparison and robustness.

The machine-learning pipeline is intentionally lightweight and explainable. It performs column normalization, mixed-format datetime parsing, rare-category grouping, age conversion to numeric day counts, and calendar feature extraction from intake timestamps. Two baseline models are used: Logistic Regression and Random Forest. On the verified full-data stratified split for the main `fast_adoption_30d` task, Logistic Regression reaches Accuracy 0.6463, F1-score 0.5828, and ROC-AUC 0.7313, while Random Forest reaches Accuracy 0.7289, F1-score 0.6406, and ROC-AUC 0.8149. This makes Random Forest the strongest baseline in the main setting.

Two comparison experiments strengthen the project package. First, the fallback target `adoption` is easier to predict than the 30-day target, with Random Forest reaching Accuracy 0.7156, F1-score 0.7268, and ROC-AUC 0.7961. Second, a time-based split for `fast_adoption_30d` produces lower performance than the default stratified split, with Random Forest reaching Accuracy 0.6983, F1-score 0.5773, and ROC-AUC 0.7654. This suggests the time-based setting is a stricter and more realistic evaluation.

To make the repository feel complete as a final course project, the materials package also includes a subgroup analysis by animal type and a lightweight random-forest ablation without `intake_year`. The subgroup analysis shows meaningful differences between dogs and cats in both descriptive rates and subgroup test performance. The ablation reduces Random Forest performance only modestly, indicating that `intake_year` is useful but not the only important signal. The strongest limitation remains the chronological pairing heuristic for repeated `Animal ID` records, which is intentionally simple and explainable rather than fully reconstructive.
